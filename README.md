# Docker Nginx Virtual Hosts

Було створено другий контейнер nginx та запущено на порту 8080

Налаштовано два віртуальні хости:
- my.test.loc
- api.test.loc

Для контейнера примонтовано:
- папку з конфігураціями nginx
- папку з файлами сайтів

Перевірка:
- http://my.test.loc:8080
- http://api.test.loc:8080
